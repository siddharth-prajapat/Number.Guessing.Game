# 🎯 Guess The Number!

A simple and fun **Number Guessing Game** built using HTML, CSS, and JavaScript. The player has to guess a number between 1 and 100 within a limited number of attempts.

## 🔴 Live Demo

👉[![Demo](https://img.shields.io/badge/Demo-%231877F2.svg?logo=Demo&logoColor=white)](https://siddharth-prajapat.github.io/Number.Guessing.Game/)


---

## 🚀 Features

- 🎮 Pixel-style retro UI design
- 🔢 Limited attempts system (Max: 3 attempts)
- 📊 Real-time attempt counter
- ✅ Win/Lose messages
- 🔄 Restart button to play again
- 🔊 Sound effects for win, wrong guess, and game over
- 📱 Mobile and tablet responsive design

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)

## ▶️ How to Play

1. Open the 👉 [![Demo](https://img.shields.io/badge/Demo-%231877F2.svg?logo=Demo&logoColor=white)](https://siddharth-prajapat.github.io/Number.Guessing.Game/)
2. Enter a number between **1 and 100** in the input box
3. Click **Check!** to submit your guess
4. You have a maximum of **3 attempts** to guess the correct number
5. Win 🏆 or lose 🥲 — click **Restart** to play again!

---

## 📦 Run Locally

```bash
git clone https://github.com/siddharth-prajapat/Number.Guessing.Game.git
```
